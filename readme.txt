== Description ==

Tsubaki puts the spotlight on your products and your customers.  This theme leverages WooCommerce to provide you with intuitive product navigation and the patterns you need to master digital merchandising.

== Changelog ==

= 1.3.0 =
* Introduced blockified templates for the single product, the archive(shop), and the product results page.
* Fixed a styling issue affecting full-width elements in the site editor.

= 1.2.2 =
* Fixed an issue that prevended editing buttons within patterns.

= 1.2.1 =
* Tweaked the home page template for global styles support
* Styling improvements on supporting the blockified single product template

= 1.2.0 =
* Changed theme base from TT2 to TT3
* Modified account navigation width and styling
* Removed default button borders for better global styles support
* Added form inputs styling to match the look and feel of the ones in the cart and checkout blocks
* Tweaked the default table styling in thank-you and my-account pages
* Tweaked product tabs styling for better global styles support
* Fixed the global store notice styling

= 1.0.1 =
* Fix load order for WooCommerce compatibility

= 1.0.0 =
* Initial release
